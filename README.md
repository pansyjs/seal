<h1 align="center">Pansy Seal</h1>

<div align="center">
  强大的印章组件，帮你快速的生成印章。
</div>


## 🏗 安装

```
// npm
npm install @pansy/seal --save

// yarn
yarn add @pansy/seal

// pnpm
pnpm install @pansy/seal
```

## 🔨 使用
```ts
import { Seal } from '@pansy/seal';

const seal = new Seal({ ... });

// 如果需要修改印章参数，请调用
seal.update({ ... });

// 销毁印章
seal.destroy();
```
